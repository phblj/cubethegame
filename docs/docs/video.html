Video #

A react component for displaying 2D video. Supported format is WebM.

Note: Auto-play is restricted on some browsers. For example on Chrome Android, auto-play only works if the video is muted, otherwise a user gesture (typically touch, mouse, and keyboard but not gamepad) is required to begin video playback.

Example Usage:

<Video style={{width: 3.0, height:2.0}} source={{uri: 'assets/my-video.webm'}} />

Props #

View props... #

autoPlay PropTypes.bool #

Whether video starts playing automatically when component is loaded. Auto-play is restricted on some browsers due to security considerations. Default is true

muted PropTypes.bool #

Whether the video is muted. Default is false

playStatus PropTypes.oneOf(['play', 'pause']) #

Controls the playback status. If not set, the value of autoPlay determines whether the video plays when the component is loaded.

source PropTypes.shape({uri: PropTypes.string}) #

source video in the form of {uri: 'http'}
